# Robot Algorithm (C++)
mobile robot algorithm samples

## Denpendency
* Python3
* matplotlib

---

## Setup

### git clone this package
```bash
git clone https://github.com/hrjp/robot_algorithm
```

### install matplotlib
```bash
sudo apt install python3-matplotlib python3-numpy
```
